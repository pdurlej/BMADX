# PRD — BMADX v0.2.3

## Goal

Move BMADX from a public working repo for insiders to a public repo that
non-technical, low-friction Codex users can actually adopt.

## Product goals

- public surfaces in English
- portable install and remediation flows
- onboarding that ends in a first useful task
- proof surfaces closer to user experience than token tables alone

## Scope

- translate public skill/docs/templates/sample bundle
- remove maintainer-local path leakage
- add install-and-verify wrapper
- add `START_HERE`, quickstart, prompt pack, chooser, FAQ, and Rescue Mode guide
- add transcript-based proof surfaces

## Success criteria

- a new user can install BMADX and get to a first useful task quickly
- public docs no longer assume gear-level framework familiarity
- runtime output and public samples are portable
- BMAD remains the process source of truth
